# Stream Proto
- Install npm into the project
  ``` 
  npm install
  ```
- Install Polymer into the project
  ``` 
  npm install -g polymer-cli
  ```
- Install Bower into the project
  ``` 
  npm install bower
  ```
  
### Viewing Application
```
npm run serve
```

# Push to Firebase
```
firebase install
```
```
firebase depoly
```
